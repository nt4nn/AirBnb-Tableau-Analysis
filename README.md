# Airbnb Data Analysis (Tableau)

## Project Overview
This project analyzes Airbnb listing data to uncover insights about pricing, demand, and host performance.

## Key Insights
- Prices vary significantly based on room type and neighbourhood
- Certain neighbourhoods dominate in terms of demand
- Highly reviewed listings indicate higher popularity
- Availability impacts pricing strategy

## Files Included
- Airbnb Project Tableau.twbx → Tableau dashboard file
- Dashboard screenshots → Visual insights

## Tools Used
- Tableau
- Data Visualization
- Data Analysis

## Key Features
- Interactive dashboards
- Pricing analysis
- Host performance tracking
- Location-based insights

## How to Use
Download the .twbx file and open it in Tableau Desktop to explore dashboards.
